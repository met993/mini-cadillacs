# mini-cadillacs

A simplified version of a side-scrolling "beat 'em up" game inspired by Cadillacs and Dinosaurs. Built with Python and Pygame, this project offers a minimal yet expandable framework for starter game development, prototyping, and educational use.

## Features

- Playable hero character with basic movement and attack
- Simple enemy AI and collision detection
- Score and health system
- Expandable game structure for customization

## Installation

1. Clone the repository:

    ```
    git clone https://github.com/yourname/dino-fighter-simple.git
    cd dino-fighter-simple
    ```

2. Create and activate a virtual environment (optional but recommended):

    ```
    python -m venv venv
    # Windows:
    venv\Scripts\activate
    # macOS/Linux:
    source venv/bin/activate
    ```

3. Install dependencies:

    ```
    pip install -r requirements.txt
    ```

## Usage
Run the game with:

python main.py

Use the keyboard to move, jump, and attack. Have fun fighting dinosaurs and collecting points!
